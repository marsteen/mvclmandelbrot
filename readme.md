# OpenCL Test Project

## Building mvclmandelbrot executable
Change to "project/build" folder and call:
    cmake ..
    make

### Prerequisites
* Ubuntu: ocl-icd-opencl.dev
* Windows: AMD AMP Sdk
* Mac OS/X: Xcode


### Test run
from the folder "project" run script:
    run-float.sh
or:
    run-double.sh
