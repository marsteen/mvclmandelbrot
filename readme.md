# OpenCL Test Project

## Building the mvclmandelbrot executable
Change to "project/build" folder and call:<br>
```
cmake ..
make
```

### Prerequisites
* Ubuntu: ocl-icd-opencl.dev
* Windows: AMD AMP Sdk
* Mac OS/X: Xcode


### Test run
from the "project" folder run script:<br>
&nbsp;&nbsp;```./run-float.sh```<br>
or:<br>
&nbsp;&nbsp;```./run-double.sh```<br>
